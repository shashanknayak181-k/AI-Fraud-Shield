# AI Fraud Shield

## Overview

AI Fraud Shield is an AI-powered web application designed to help users detect and prevent digital fraud. It analyzes suspicious SMS messages, phishing URLs, and currency images using Google's Gemini AI, providing accurate classifications along with clear explanations.

---

## Features

- 📩 AI-powered SMS Scam Detection
- 🔗 Phishing URL Detection
- 💵 Currency Verification
- 🤖 AI-generated explanations
- 🌐 Responsive web interface
